BINGO

To use the Bingo Questions Generator, start by locating the "Enter Questions" section on the webpage. In the text area provided, input your questions. Each question should be on a new line.

After entering all your questions, click the "Preview PDF" button below the text area. The application will process your input and generate a preview of the PDF. If you are satisfied with the preview, you can download the PDF file by using the download button located on the upper right corner, beside the print button. Use the "Back" button to return to the previous page.

To use the Bingo Card Generator, start by selecting the matrix size (3x3, 4x4, or 5x5) by clicking the corresponding button. Enter the number of pages you want to generate in the "Number of Pages" input field.

Fill each cell in the matrix with your custom text. Once you’ve filled in all the cells, click the "Preview PDF" button to generate a preview of your Bingo cards. The PDF preview will appear on the right side of the screen.

To save the PDF, use the download button located on the upper right corner, beside the print button. If you need to start over, click the "Reset" button to clear the inputs. Use the "Back" button to return to the previous page.

For example, you might choose a 3x3 matrix and enter 3 pages. Fill in the input fields, click "Preview PDF," and download the PDF from the preview.

WHEEL

To use the Wheel of Names, start by entering your choices in the provided text area under "Enter Choices." Each choice should be on a new line. Once you have entered all your choices, click the "SUBMIT" button to generate the wheel with your entries. Click the "spin" button and when a choice has been selected, a popup window will appear displaying the chosen entry. You will have the option to either remove this entry from the wheel by clicking the "Remove" button or keep it on the wheel by clicking the "Keep" button.

To clear the text area and remove all entries, click the "CLEAR" button. If you want to return to the home page, click the "HOME" button.

HUNT

To use the Word Hunt generator, start by entering your words in the "Enter words" text area. Ensure each word is on a new line. Next, specify the grid size (between 5 and 20) in the "Enter grid size" input field. The default grid size is set to 10.

Once you have entered the words and chosen the grid size, click the "Preview PDF" button to generate a preview of your Word Hunt puzzle. The PDF preview will be displayed on the right side of the screen. To save the PDF, use the download button located on the upper right corner, beside the print button. If you want to return to the home page, click the "HOME" button.

PASCAL

To use the Pascal's Triangle Solver, enter a number between 1 and 25 in the input field provided. Click the "Generate Triangle" button to see the Pascal's Triangle along with the binomial expansion of (x + y) raised to the power of n-1. Click the "HOME" button to return to the main page.

JEOPARDY

To edit questions and answers, start by entering a category title in the provided input field. Each category should have multiple clues (questions and answers). Initially, three fields are provided for entering clues. For each clue, fill in the question, the correct answer, and three decoy answers. To add more clue fields, click the "+" button next to the last clue field. To remove a clue field, click the "–" button within the specific clue field you want to remove. Once you have filled in the category title and all the clues, click the "Submit" button to add the new category to the saved categories list. The newly added category will appear in the list below the form.

To edit an existing category, click the "Edit" button next to the respective category in the saved categories list. This will populate the form with the existing data, allowing you to make changes. After making the necessary changes, click the "Save Changes" button to update the category. To delete a category, click the "Delete" button next to the respective category in the saved categories list. This will remove the category from the list and the local storage. Finally, click the "BACK" button to return to the main Jeopardy page.

For the Multiplication Choice type, if data is stored, the game board will display automatically. Click on a clue under your chosen category to see the question and answer choices. Select an answer to see if you are correct. After clicking, the screen will show whether your answer is correct or not. Then, it will redirect you to the gameboard and your score updates accordingly. Click the "BACK" button to return to the game type selection page at any time.

For the Identification type, if data is stored, the game board will display automatically. Click on a clue under your chosen category to see the question and answer choices. Type your answer in the input field and click the "Answer" button to submit. The screen will show whether your answer is correct or not. The modal displaying the result will disappear after a short period, returning you to the game board, and your score will update accordingly. Click the "BACK" button to return to the game type selection page at any time.
